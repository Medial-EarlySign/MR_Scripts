medbuild_tools - build all tools for external use (on remote computers and client computers)
medbuild_external_resources - docker that download bootstrap and other libs for medial projects compilation (can only be built on external computer with internet connection)
medbuild_external_pyenv - docker that builds our entire python development environment (can only be built on external computer with internet connection)
centos1810dev - docker that enables basic development tools and some lib headers required later by compilation.
